## 合作

### 校企联合培养

谁主导？以企业的人力资源管理需求为中心构建。

面向谁？线上兼职实习的在校生。

如何分工？企业为主，学校根据企业需求讲不好在工作中教的，比如经管原理课，数理课等。

## 激励

构建互助文化、明确权责、锻炼讲课等等。

把公司承担教学成本作为绩效给新人，公司出钱给老人带新人。
